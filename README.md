# nautisaimaa
Mobile app with nautical maps of the Saimaa area.


Planned features:
  - offline nautical maps of Finland
  - route drawing, saving and distance calculation
  - locking to compass

Possible features:
  - location sharing
  - additional map types
  - speed meter


| Tech used |
| --------------:|
| react |
| webpack, babel |
| cordova |
| html, js, css |
| openlayers3 |

To run:
```sh
$ cordova platform add android
$ cordova run android
```
